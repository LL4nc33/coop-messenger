# Security Policy

## Supported Versions

We currently support the latest version of Coop. Please make sure you stay up-to-date.

## Reporting a Vulnerability

Please report security vulnerabilities privately via [GitHub Security Advisories](https://github.com/LL4nc33/coop-messenger/security/advisories/new).

Alternatively, you can open a [GitHub Issue](https://github.com/LL4nc33/coop-messenger/issues) with the `security` label if the vulnerability is not sensitive.

**For vulnerabilities in the underlying ntfy engine**, please report directly to the [ntfy project](https://github.com/binwiederhier/ntfy/security).
